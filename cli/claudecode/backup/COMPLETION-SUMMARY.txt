================================================================================
 SUPPORT & TROUBLESHOOTING
================================================================================

Common Issues & Solutions:

Issue: "Permission denied" errors
  → Close all IDEs, terminals, and npm processes before backup

Issue: npm/Python not found
  → Verify installation and PATH configuration
  → Add to system environment variables if needed

Issue: Backup is too slow
  → Close unnecessary applications
  → Ensure adequate disk space (10 GB minimum)
  → Consider using -SkipNodeModules or parallel processing

Issue: Backup size too large
  → node_modules not included by default (expected)
  → package-lock.json can be 5-50 MB per project
  → Use compression (automatic)

Issue: Restore fails
  → Run with -ValidateOnly first to check backup integrity
  → Verify backup path is correct
  → Check RESTORE_REPORT.json for detailed error messages

For detailed troubleshooting, see DEV-ENVIRONMENTS-README.md section "Troubleshooting"
